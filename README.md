# adventofcode-rust-2022
Solving https://adventofcode.com/2022 in rust

Run all tests:
```
cargo test --release
```

Run tests (example input) for a single day:
```
cargo test --release --bin day01
```

Run real problem input for a single day:
```
cargo run --release --bin day01
```
